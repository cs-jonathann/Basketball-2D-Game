# 2D Basketball Game

A Java 1.8 2D basketball game built with Applets.

## Features
- Player vs CPU 1-on-1 gameplay
- Shot meter with timing-based green window
- CPU AI with offensive (roaming) and defensive states
- Block, steal, and rebound mechanics
- Backboard and rim physics
- 2 and 3 point scoring

## Controls
| Key | Action |
|-----|--------|
| WASD 	   | Move |

| C + WASD | Run |

| O (hold) | Charge shot, release in green window |

| M | Reach / steal |

| SPACE | Jump (block) |

| P | Pick up / drop ball |

## Built With
- Java 1.8
- Applets (GameBase framework provided by professor)
